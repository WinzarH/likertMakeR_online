## <img alt = "LikertMakeR logo" src="www/LikertMakeR_4.png" width = 10% LikertMakeR_online 

A Shiny web app for generating **synthetic Likert-style survey data** from user-specified summary inputs.

This app is a cut-back GUI version of the **LikertMakeR** R package. 

It lets you:
  - create or upload a **correlation matrix** (including a “target Cronbach’s alpha + #items” workflow),
  - specify **means, SDs, bounds, sample size, and items-per-scale**,
  - generate a synthetic dataset that matches those target parameters,
  - **validate** results with visual and statistical diagnostics,
  - download outputs as CSV.

> Note: This app generates **plausible synthetic data** consistent with your inputs. It **cannot** recover original raw data.
